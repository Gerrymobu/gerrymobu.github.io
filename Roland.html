Roland Garros

The most prestigious tennis tournament in France intersects with the realm of data science, illuminating how this discipline serves as a valuable tool for analyzing tournament information, results, and matches played since 1970. Leveraging this extensive database, a CRISP-DM methodology was employed to develop a predictive model aimed at maximizing accuracy.

Our database underwent a thorough cleansing process to retain only the pertinent variables for the matches to be predicted. Subsequently, a new variable, referred to as 'prediction,' was created. This variable serves as the target for prediction, with the values '1' indicating Player A's victory and '0' denoting Player B's victory.

En nuestra base que servirá de entrenamiento se creo esta variable con el peso de el ranking del jugador y los puntos. Es importante mencionar que en la base de datos a predecir se encontraban datos faltantes por lo que se manipularon estos datos con un nueva variable que diera mas peso al jugador del que mas dato se contaba y que le favorecían mas para ganar.

Una vez hecho esto usamos la herramienta de promidat la cual se llama predictor para introducir nuestros datos y sacar una correlación de las variables y después probar diferentes modelos hasta tener el mas adecuado

Here we present a comparison of all models, and the one chosen for implementation is the Lasso penalized regression. This model exhibited the highest indices of '1's, which is of utmost importance as they indicate the winning player.

Once the model to be used was determined, we proceeded to run it alongside our training data in the predictor tool. After generating and running the model on our database for prediction, it yielded the results displayed in the attached Excel file. These results were then interpreted to visually identify the winner's name.

Having as final result 80% of correct estimated match results

This project underscores the potent synergy between data science methodologies and sports analytics, as exemplified by our exploration of the French tennis tournament dataset. Through meticulous data cleaning, model selection, and predictive analysis, we have demonstrated the efficacy of machine learning techniques in forecasting match outcomes. The adoption of the Lasso penalized regression model further solidifies our commitment to precision and accuracy, yielding insightful predictions for player performance. Moving forward, this project not only contributes to our understanding of tennis dynamics but also highlights the transformative potential of data-driven approaches in sports analytics.
